## 〇、設計概念

視覺語言來自「企劃人的工作素材」：提案書、時程表、螢光筆畫重點、紅筆改稿。

- **螢光筆黃是全站唯一的視覺記憶點**，只用在關鍵字與成果數字（`<mark>` 標籤），不可濫用
- 等寬字型（IBM Plex Mono）用於日期、編號、eyebrow，營造「專案時程表」感
- 案例採 `CASE 01`、區塊採 `01／SELECTED CASES` 的提案編號式標記
- **內容第一眼可見**：不使用翻轉卡、hover 才展開、預設收合等把內容藏起來的互動
- 資訊順序為「招募邏輯」：定位 → 證明（案例）→ 可信度（經歷）→ 能力 → 差異化（AI 工具）→ 個性（關於我）→ 轉換（聯絡）

## 一、色彩系統（CSS 變數）

```css
:root{
  --paper:#fbfaf8;          /* 紙白：主要背景 */
  --paper-soft:#f3f1ec;     /* 次要背景（案例、AI 工具區） */
  --ink:#1d2732;            /* 墨青：標題文字、深色區塊背景 */
  --ink-elevated:#28323e;   /* 深色卡片背景 */
  --body:#3c434b;           /* 內文文字 */
  --muted:#79808a;          /* 次要文字、說明文字 */
  --hl:#ffe14d;             /* 螢光筆黃：唯一強調色 */
  --hl-soft:rgba(255,225,77,.55);  /* 螢光筆淺色（淺底 mark 用） */
  --pen:#c8452a;            /* 紅筆色：僅用於連結與 focus 外框 */
  --line:#dcd8cf;           /* 分隔線、卡片邊框 */
}
```

**用色規則：**
- 螢光筆黃只出現在：`<mark>` 標記、時間軸節點與線、計數器數字、LIVE 狀態章、聯絡區主按鈕
- 紅筆色只出現在：文字連結、`RESULT` 標籤、focus-visible 外框、進修區小 icon
- 深色區塊（工作經歷、聯絡我）內的 `<mark>` 使用透明度較低的黃（rgba(255,225,77,.28)）

## 四、RWD 斷點

| 裝置 | 斷點 | 行為 |
|---|---|---|
| 桌機 | >960px | 能力／工具 3 欄、案例左圖右文、關於我文＋照片兩欄 |
| 平板 | ≤960px | 能力／工具 2 欄、案例改上圖下文、關於我照片移到文字前 |
| 手機 | ≤640px | 全部單欄、經歷數字直排、漢堡選單（點連結自動收合） |

**無障礙與動效規則：**
- 尊重 `prefers-reduced-motion`：關閉捲動浮現與計數動畫，直接顯示最終狀態
- `:focus-visible` 使用紅筆色外框
- 燈箱支援 Esc 關閉、左右方向鍵切換、點背景關閉

## 五、各區塊設計規格

### Hero
- mono 小字 `PORTFOLIO／2026` → 姓名（serif 900）＋ mono 英文副標
- **定位宣言（thesis）**：serif 700，兩行，關鍵詞用 `<mark>` 標螢光筆
- 一段補充說明 → 兩顆 CTA（實心墨青「看精選案例」＋ 描邊「與我聯絡」）
- 底部三個事實數字（mono）：+278.5% 社群觸及成長、100+ 工作坊、10 年+ 跨產業；每個數字為卡框式（`--line` 邊框、圓角 8px、hover 微浮起），手機改單欄直排
- ❌ 不使用打字機效果與關鍵字雲（v1 元素，已移除）

### 精選案例（#cases）
- 直列卡片，桌機為左圖（340px）右文
- 卡片結構固定：`CASE 0N／分類`（mono）→ 標題 → 公司・角色 → 描述 → **RESULT 行**（紅筆 mono 標籤＋成果句，數字用 `<mark>`）→ 標籤列
- 圖片區點擊開燈箱，右下角 mono 標示張數；圖片載入失敗時 `onerror` 移除、顯示佔位
- 封面為**固定 680:420 比例的圖版**（`aspect-ratio:680/420`）：桌機內嵌卡片左欄（margin 26px、圓角 6px、垂直置中），平板以下滿寬置頂同比例；❌ 封面不得隨卡片高度拉伸（文字較多的卡片會把封面拉成長方形、視覺不統一）
- 封面 `<img>` 使用 `object-fit: contain`（完整顯示不裁切）＋ `background: var(--ink)`（與 SVG 底色相同）；❌ 不用 `object-fit: cover`（會裁到封面文字）

**CASE 01 為全寬 Featured 案例**：無側圖，內含「挑戰／策略／執行」敘事＋四格成效數字（`.case-stats`），數字資料不可改動：56.7 萬（▲278.5%）、63.7 萬（▲38.2% YoY）、10 萬+（好好玩系列）、2,259+（148 次分享）。

**案例封面一律使用 SVG 插畫**（images/cover_case0N_*.svg，墨青底＋紙白線稿＋螢光黃點睛的「章節扉頁」風格），❌ 不得用實際照片當卡片封面（長寬比不一會破版）；照片一律放燈箱。cover_case01_mit.svg 不在卡片上使用，保留作 OG 分享圖素材。
